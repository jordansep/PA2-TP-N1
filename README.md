# PA2-TP-N1

## Descripción del proyecto
Programación Aplicada II - Trabajo Práctico Nro 1.
Este repositorio contiene el código fuente inicial para el desarrollo del primer trabajo práctico de la materia.

## Tecnologías usadas
- C#
- .NET

## Cómo ejecutarlo
Para compilar y ejecutar el archivo principal (`main.cs`) desde la terminal, puedes utilizar el compilador de C# (`csc`):

```bash
csc main.cs
main.exe
```
*(Nota: Si en el futuro inicializas este repositorio como un proyecto estándar de .NET, podrás ejecutarlo simplemente usando el comando `dotnet run`)*
